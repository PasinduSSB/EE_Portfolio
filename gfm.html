Detailed Project

Grid-Forming Inverter Control (GFM)

Development and simulation of a grid-forming inverter using droop control to support voltage and frequency stability in low-inertia power systems.

View Projects

Key Contributions

  • Implemented droop-based GFM control using P–f and Q–V characteristics
  • Simulated islanded and grid-connected operation
  • Tested system behavior under weak, medium, and strong grid conditions
  • Evaluated performance under varying PV irradiance
  • Analyzed system response to step load and fault disturbances

Problem Statement

  • Renewable integration reduces synchronous inertia in modern power systems.
  • Grid-following inverters depend on an existing voltage and frequency reference.
  • Weak and low-inertia grids require grid-forming control for stable operation.

Methodology

  • Developed the inverter model in MATLAB/Simulink.
  • Implemented droop control loops for P–f and Q–V regulation.
  • Simulated islanded mode under constant load and step load operation.
  • Simulated grid-connected mode under weak, medium, and strong grid conditions.
  • Tested low and high PV irradiance operating conditions.
  • Performed disturbance tests using step load changes and fault conditions.

Test Scenarios

  • Islanded Mode: Constant load and step load operation.
  • Grid-Connected Mode: Weak, medium, and strong grid conditions.
  • Irradiance Variation: Low and high PV generation scenarios.
  • Disturbance Tests: Step load and fault conditions.

Measured Variables

  • Frequency response
  • Active power (P)
  • Reactive power (Q)
  • Voltage and current waveforms
  • RMS voltage
  • DC-link voltage

Control Strategy

  • P–f droop adjusts frequency based on active power output.
  • Q–V droop adjusts voltage based on reactive power exchange.
  • This control approach enables voltage-source behavior in grid-forming operation.
  • The inverter supports system stability without relying only on an external grid reference.

Results

  • Stable frequency regulation under step load variations.
  • Effective power sharing through droop characteristics.
  • Voltage maintained within acceptable limits in weak grid conditions.
  • System remained stable during fault disturbances.
  • DC-link voltage remained regulated under varying operating conditions.

Tools Used

  • MATLAB / Simulink
  • Control system modeling
