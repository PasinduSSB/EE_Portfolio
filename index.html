Detailed Project

EMS Microgrid System

Problem Statement

Renewable-powered microgrids must balance variable PV generation, changing load demand, battery state of charge, and utility grid exchange. Without a clear energy management strategy, the system can waste solar energy, increase grid imports during peak periods, and operate the battery inefficiently.

Methodology

The EMS applies a rule-based dispatch workflow over a 24-hour operating profile. PV generation first supplies the load, surplus energy charges the battery within SOC limits, and remaining surplus is exported or curtailed. During generation deficits, the battery discharges before the system imports power from the grid.

Tools Used

  • Python for dispatch logic and energy calculations
  • Streamlit for the interactive engineering dashboard
  • Pandas for hourly load, PV, SOC, and tariff data handling
  • Matplotlib or Plotly for result visualization

Results

The model estimates battery SOC behavior, grid import and export, PV self-consumption, and daily operating cost. It provides a clear comparison between renewable availability and load demand, helping evaluate storage sizing, peak import reduction, and microgrid operating performance.
